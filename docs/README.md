# 자동차 경주 게임
## 요구 사항
* [ ] 전진 가능 여부
  * 4 이상일 경우 전진 가능
* [ ] 우승자 확인 기능
* [ ] 0~9 사이의 무작위 값 반환
* 입력
  * [ ] 이름은 쉼표를 기준
  * [ ] 이동 횟수
* 출력
  * [ ] 우승자 이름
    * 여러 명일 경우 쉼표로 구분
* 검증 및 유틸
  * [ ] 이름을 쉼표로 구분
  * [ ] 이름이 6자 이상 검증
    * 이름은 알파벳 소문자로만 이루어져 있어야 한다
  * [ ] 이름이 하나인지 검증
  * 잘못된 입력 값일 경우 ```IllegalArgumentException```발생 후 종료

## 객체
* 자동차
  * 이름
  * 전진 횟수
* 우승자
  * 자동차 리스트
